# Recommendation-Manage-Front 

## XXX平台管理后台

### 前端运行指令

```bash
# 克隆项目
git clone https://XXXX.XXXX.XXXX.XXXX


# 项目根目录安装依赖
pnpm i 或运行 pnpm run init

# 以开发后台地址启动前端开发服务
pnpm run serve:dev

# 以测试后台地址启动前端开发服务
pnpm run serve:stage

# 以生产后台地址启动前端开发服务
pnpm run serve:prod

# 以开发后台地址构建打包
pnpm run build:dev

# 以测试后台地址构建打包
pnpm run build:stage

# 以生产后台地址构建打包
pnpm run build:prod

# eslint语法校验与部分自动修复
pnpm run lint:fix

# husky文件生成（已经配置好了，一般不需要运行）
pnpm run prepare

# 预览构建项目服务（直接运行打包产物，本地预览）
pnpm run preview
```
